# Multikids Legal Pages

Páginas legais públicas da Clínica Multikids.

Projeto estático em HTML e CSS puro, criado para atender aos requisitos do Meta Developers e aproveitado como página pública de política de privacidade e solicitação de exclusão de dados.

O projeto é separado do site principal e da API interna.

## Páginas

* Política de Privacidade: `politica-de-privacidade.html`
* Solicitação de Exclusão de Dados: `exclusao-de-dados.html`
